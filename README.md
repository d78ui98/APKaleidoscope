## Github guide
You can refer to these steps if you are new to github or if you want

#### install git
```
sudo apt install git
```

#### clone repo with
```
git clone 
```

#### Add a file
```
git add <FILENAME>
```

#### Commiting those changes
```
git commit -m "short message"
```

#### pushing changes online
```
git push 
```
